==========================
Population Structure Analysis
==========================
This tool provides visualization of admixture and population structure in geographic space,
which accepts data in the format typically generated by admixture analyses.

Read before use
------------------		
Users need to prepare one files, admixture data.

**Admixture data:** The data must contain Ecotype, Samples, Cluster, Value, and K columns. Data from excel (format: xlsx).

**Input example data:** `Admixture data <https://github.com/Jasonxu0109/MangroveDB/releases/download/Example/Admixture_qlist.xlsx>`_.
