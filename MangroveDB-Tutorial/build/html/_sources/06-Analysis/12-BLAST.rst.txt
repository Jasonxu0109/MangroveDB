==========================
BLAST
==========================
This tool finds homologous genes between two species.

Read before use
------------------		
Users need to prepare two files, reference sequences and query sequences.

**The reference sequences:** FASTA sequence. Data from txt (format: txt).

**The query sequences:** FASTA sequence. Data from txt (format: txt).

**Input example data:** `Reference sequences <https://github.com/Jasonxu0109/MangroveDB/releases/download/Example/Athaliana_protein.fa>`_,
`Query sequences <https://github.com/Jasonxu0109/MangroveDB/releases/download/Example/Kandelia_obovata.protein.fa>`_.
